Google Sheet 集成说明

实时行情（推荐）：在你的 Google Sheet 中创建一个 LivePrices 标签页，A列放代码，B列用 =GOOGLEFINANCE("NASDAQ:MSFT","price") 公式。连接Sheet后点「刷新行情」即可自动读取。

直接获取：仪表盘会优先尝试 Yahoo Finance API 直接拉取，无需额外配置。如果被CORS限制则自动回退到Sheet代理。

推送数据：方法一：扩展程序 → Apps Script 部署 Web App。方法二：「导出」JSON 手动粘贴。方法三：配合 wang_investment_monitor_v4.gs。
